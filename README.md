# Labaratory-work2
Labaratory work 2, made by Koval Yevhenii, PZPI-25-1
E-Kahoot — Платформа для інтелектуальних змагань
Опис проєкту
E-Kahoot — це програмна система для проведення швидких інтелектуальних змагань у реальному часі. Основна мета системи — надати користувачам можливість створювати вікторини та змагатися з друзями за допомогою ігрових кімнат і динамічних таблиць лідерів.

---
##1. Функціональні вимоги (FR)
Повний список вимог доступний у файлі: [requirements.md](requirements.md).

| ID | Опис вимоги |
| :--- | :--- |
| **FR-01** | Система повинна відображати стрічку фактів для користувача. (Must) |
| **FR-02** | Система повинна генерувати миттєвий квіз після кожного прочитаного факту. (Must) |
| **FR-03** | Система повинна нараховувати користувачу бали за правильні відповіді. (Must) |
| **FR-04** | Система повинна зберігати дані профілю та історію пройдених квізів. (Must) |
| **FR-05** | Система повинна дозволяти вхід за допомогою зовнішнього API аутентифікації. (Must) |
| **FR-06** | Система повинна підтримувати режим "батл" для змагань між користувачами. (Should) |
| **FR-07** | Система повинна забезпечувати функції соціальної взаємодії. (Should) |
